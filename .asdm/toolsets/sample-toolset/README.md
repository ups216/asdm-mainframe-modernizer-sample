# Sample Toolset

This is a sample toolset demonstrating the structure and components of an ASDM toolset.

## Overview

The sample toolset provides example implementations of actions, specifications, and tools that can be used as a reference for creating new toolsets.

## Components

### Actions
Actions are converted to slash commands for AI coding tools. See the `actions/` directory for examples.

### Specs
Specifications provide rules and guidelines for the toolset. See the `specs/` directory for examples.

### Tools
Tools are utilities designed for CLI environments. See the `tools/` directory for examples.

## Installation

See [INSTALL.md](INSTALL.md) for installation instructions.

## Usage

1. Review the available actions in the `actions/` directory
2. Check the relevant specifications in the `specs/` directory
3. Use the provided tools as needed for your development workflow
