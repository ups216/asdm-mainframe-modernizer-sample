# Action Tool 001

## Description
This is an example action that demonstrates how to create slash commands for AI coding tools.

## Usage
```
/action-tool-001 [options]
```

## Parameters
- `--param1`: Description of parameter 1
- `--param2`: Description of parameter 2

## Examples

### Basic usage
```
/action-tool-001 --param1 value1
```

### Advanced usage
```
/action-tool-001 --param1 value1 --param2 value2
```
