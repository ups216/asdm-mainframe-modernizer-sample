# Integrate Services

## Description
Wire up microservice communication, API gateways, service discovery, and shared data access patterns in the modernized banking application. This action connects all individual services into a cohesive, integrated system.

## Usage
```
/asdm-integrate-services
```

## Process
1. Configure API Gateway routing and request filtering
2. Set up service discovery and registration (Spring Cloud / Kubernetes Services)
3. Implement inter-service REST client configurations
4. Configure shared database access and transaction management
5. Set up messaging/async communication where needed
6. Implement circuit breakers and resilience patterns
7. Configure security, authentication, and authorization

## Output
- API Gateway configuration
- Service discovery setup
- Inter-service communication configurations
- Security configuration
- Resilience patterns (circuit breakers, retries)
